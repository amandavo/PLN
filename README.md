## Como executar

1. Clone o repositório:
  ```
  git clone https://github.com/amandavo/PLN.git
  ```
2. Entre nas pastas:
  ```
  cd PLN
  cd tc1_1
  ```
3. Instale o leitor de pdf:
  ```
  pip install pymupdf
  ```
4. Rode o programa:
  ```
  python count.py
  ```
